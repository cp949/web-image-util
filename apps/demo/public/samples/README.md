# Demo Samples

데모 앱이 사용하는 정적 자산이다. 모두 `apps/exam/public/sample-images/`에서 회수한 파일이다.

| 파일 | 원본 (apps/exam) | 크기/특성 | 사용 데모 |
| --- | --- | --- | --- |
| `photo.jpg` | `sample4.jpg` | 500×439 사진 | 빠른 시작 / Presets / Shortcut / 체이닝 |
| `landscape.jpg` | `sample1.jpg` | 2560×1600 사진 | Fit 모드 비교 |
| `svg-icon-small.svg` | `sample1.svg` | 91×114 단색 path | SVG 처리 (확대 시연) |
| `svg-character.svg` | `sample2.svg` | 89×114 defs+inline style | SVG 처리 (변형용) |
| `svg-illustration-wide.svg` | `sample3.svg` | 558×334 gradient | (예비) |
| `svg-illustration-alt.svg` | `sample4.svg` | 558×334 gradient | (예비) |

라이센스는 apps/exam의 원본과 동일하다. apps/exam이 제거되면 본 폴더가 새로운 단일 출처가 된다.
